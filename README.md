# JavaINSA

2D Java platformer with custom assets, audio, level progression, and game states.

## Overview

The project contains:

- gameplay logic
- input handling
- UI and menu states
- level, enemy, item, and audio handlers

## Run

```bash
./run.sh
```

If you prefer manual execution, compile and run the `Main.java` entry point with the JDK you use for the project.
